Array

Array is a collection of items. Unlike other programming language, array in javascript allow you to have different type of data and also length of the array is not fixed. Since the length of the array is not fixed, data can be stored in non contiguous locations.

Array cannot use string as indexes but must use integers. Index in array starts from 0 and goes till n-1 if the array size is n.

Array allow to have ordered collection of items which is not the case in case of objects. In case of objects we cannot maintain the order of the items. So we also use array where order is important.

There are two syntaxes for creating an empty array:

let arr = new Array();

let arr = [];

Most of the time we use the second notation.

let fruits = ["Apple", "Orange", "Plum"];

We can access array elements by its index number in square brackets where 0th index will give first array element and (n-1)th element will give last element.

let fruits = ["Apple", "Orange", "Plum"];

alert( fruits[0] ); // Apple

alert( fruits[1] ); // Orange

alert( fruits[2] ); // Plum

We can replace an array element by simply assigning the element inside square bracket notation.

fruits[2] = "Mango" ;

alert( fruits[2] ); // This will now give "Mango" as output

We can also add new item in some position by putting index inside bracket notation.

fruits[3] = "Banana" ;

We can also show the entire array by putting the array name inside output function

alert( fruits ); // ["Apple","Orange","Mango","Banana"]

Array in javascript can hold different types of data.

let Student = ["Gopal", "CSE", 2015, true] // name, branch, year, boolean

alert( Student ); // Output: ["Gopal", "CSE", 2015, true]

There are some in-built functions which can be used on array objects to insert and delete elements from array. two functions push() and pop() through which we can achieve stack on array. push() is used to insert and pop() is used to delete element at the end of the array.

let fruits = ["Apple", "Orange", "Pear"];

alert( fruits.pop() ); // remove "Pear" and alert it

alert( fruits ); // Apple, Orange

fruits.push("Mango");

alert( fruits ); // Apple, Orange, Mango

There are another two functions shift() and unshift() by which we can delete/insert element at the beginning of the array. where shift() is used to extract element from the beginning and unshift() is used to insert element at the beginning. by using shift() and push() we can achieve queue on array.

let fruits = ["Apple", "Orange", "Pear"];

alert( fruits.shift() ); // remove "Apple" and alert it

alert( fruits ); // Orange, Pear

fruits.unshift("Mango");

alert( fruits ); // Mango, Orange, Pear
