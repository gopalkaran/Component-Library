Function

Function is the one of the functional building blocks in Javascript. Function is nothing but a procedure. A procedure is set of statements to perform a certain tasks. But for a procedure to qualify as function, it should take some input and gives some output.

Function definition consists of a function keyword followed by:

  • function name.
  • A list of parameters to the function.
  • Javascript statements that define the function, enclosed in curly brackets.

The following code defines a simple function:

function square(number) {

return number * number;

}

The above function takes one parameter number. It consists of one statement that returns the parameter of the function multiplied by itself

Primitive parameters(such as number) are passed to functions by value, the value is passed to the function, if the function changes the value of the parameter, this changes not relfected globally or in the calling function

But if an object is passed as a parameter and the function changes object's property, that change is visible outside the function, as shown int he following example

function myFunc(theObject) {

theObject.make = 'Toyota';

}

var mycar = {make: 'Honda', model: 'Accord', year: 1998};

var x, y;

x = mycar.make; // x gets the value "Honda"

myFunc(mycar);

y = mycar.make; // y gets the value "Toyota"
